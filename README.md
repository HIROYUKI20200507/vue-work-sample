# vue-work-sample

## version

`node v16.14.0`

`vue v3.2.13`

`vuex v4.0.0`

## 利用ライブラリ

`vuex`

`axios`

`vuedraggable`

モック API（https://httpbin.org/）
※初期設定は https://httpbin.org/post で設定

## 初回実行コマンド

```
yarn
```

### localhost 立ち上げ

```
yarn serve
```

http://localhost:8080/

### ビルドコマンド

```
yarn build
```
